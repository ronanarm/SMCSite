Stage Plot Designer Instructions

  • Add equipment: Click on any item in the equipment library to add it to the canvas.
  • Move equipment: Click and drag items to position them on the stage.
  • Select equipment: Click on an item to select it and show its controls.
  • Resize equipment: Use the + and - buttons to increase or decrease the size.
  • Rotate equipment: Use the rotation buttons to rotate items left or right.
  • Add text labels: Click the text icon to add a label to an item.
  • Delete equipment: Drag an item to the trash icon in the bottom-right corner.
  • Adjust canvas: Change the width and height of the canvas using the controls above it.
  • Enable grid: Check the "Show Grid" option to display a grid for easier alignment.
  • Save your work: Click the "Save Plot" button to save your stage plot locally.
  • Load saved work: Click the "Load Plot" button to restore your saved stage plot.
